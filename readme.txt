********** offline-wiki-firefox-addon ***************

But: développer une extension pour firefox, facilitant la navigation dans un
ensemble d'articles extraits de wikipedia.

Auteurs: L'équipe Miftah du club open source de l'ENSIAS.

Description:
    L'extension va offrir une page d'accueil comportant des liens organisés en
    catégories vers les articles disponible.
    On pourra aussi proposer une fonctionnalité de recherche par mot clé dans
    les articles disponibles.
